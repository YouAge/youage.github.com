豆瓣插件的使用

  • 根目录下安装 npm install hexo-douban --save
  • 再根目录下配置_config.xml 文件
  • 插件地址
  • 启动命令:hexo clean && hexo douban -bgm && hexo g && hexo s 注意其中开启hexo-douban的命令中，-bgm代表的是book、game、movie三个参数，如果只需要其中的一部分就只带你想要的那些参数
